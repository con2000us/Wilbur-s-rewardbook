# 🎉 歡迎使用 Wilbur's Reward Book！

> **完全免費的學生成績追蹤系統，10 分鐘就能開始使用！**

## 🌟 這是什麼？

**Wilbur's Reward Book** 是一個專為**老師、家長、補習班老師**設計的學生成績管理系統。你可以用它來：

- 📝 **記錄學生成績** - 考試、作業、小考、專題等各種評量
- 💰 **管理獎勵系統** - 自動計算獎金，追蹤學生累積的獎勵點數
- 📊 **列印學習記錄** - 一鍵生成漂亮的學習記錄報表
- 📚 **多科目管理** - 為每個學生管理多個科目的成績
- 🌐 **雙語支援** - 支援繁體中文和英文

## 💡 使用情境

### 情境 1：家長記錄孩子成績
> 小明今天數學考試得了 95 分，系統自動計算獎金 10 元，累積到他的獎勵帳戶中。媽媽可以隨時查看小明的學習記錄，並列印出來給老師看。

### 情境 2：補習班管理多個學生
> 補習班有 20 個學生，每個學生都有不同的科目。老師可以快速記錄每個學生的成績，系統會自動計算獎金，並生成學習記錄報表。

### 情境 3：老師追蹤學習進度
> 老師可以記錄每次考試、作業的成績，系統會自動統計學生的學習表現，並可以列印成報表給家長看。

## 🚀 快速開始

**只需要 3 個步驟，10 分鐘就能開始使用！**

1. **註冊 3 個免費帳號**（每個 2 分鐘）
   - GitHub 帳號

   - 點擊「Deploy」按鈕
   - 填入資料庫資訊
   - 等待部署完成

3. **開始使用**（1 分鐘）
   - 設定密碼
   - 開始記錄學生資料

## 🎁 主要功能

### 📚 學生管理
- 添加多個學生
- 為每個學生設定頭像和顏色
- 快速切換查看不同學生的資料

### 📝 成績記錄
- 記錄各種評量（考試、作業、小考、專題）
- 自動計算分數和百分比
- 追蹤評量狀態（即將到來、已完成、已評分）

### 💰 獎勵系統
- 自動計算獎金（滿分 30 元、90 分以上 10 元、80 分以上 5 元）
- 自訂獎勵規則（可針對不同科目、不同學生設定）
- 追蹤獎金獲得和支出記錄
- 查看獎金餘額

### 📊 報表列印
- 一鍵生成學習記錄報表
- 可選擇月份和科目篩選
- 漂亮的列印格式

### 🌐 多語言支援
- 繁體中文
- English

### 💾 資料備份
- 匯出資料為 JSON 檔案
- 匯入備份資料
- 資料庫自動備份
