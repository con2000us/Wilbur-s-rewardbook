# 💰 獎金存摺使用指南

## 📖 什麼是獎金存摺？

獎金存摺就像銀行存摺一樣，用來記錄學生的獎金收入和支出明細。

---

## 🎯 功能說明

### 記錄內容

| 類型 | 說明 | 範例 |
|-----|------|------|
| **💰 收入** | 獲得獎金 | 比賽得獎、額外獎勵、進步獎勵 |
| **🛍️ 支出** | 花費獎金 | 兌換禮物、購買文具、遊戲時間 |

### 統計資訊

存摺頁面會自動顯示：
- 💚 **總收入**：所有獲得的獎金總和
- 💔 **總支出**：所有花費的獎金總和
- 💙 **當前餘額**：收入 - 支出

---

## 🚀 如何使用

### 訪問獎金存摺

1. 進入學生頁面
2. 點擊「💰 獎金存摺」按鈕

### 新增收入記錄

#### 預設收入類型：
- 🎁 額外獎勵
- 🏆 比賽得獎
- ⭐ 優良表現
- 📈 進步獎勵

#### 操作步驟：
1. 點擊「➕ 新增記錄」
2. 選擇「💰 收入」類型
3. 可以點擊預設事件快速填寫
4. 填寫：
   - **描述**：例如「數學競賽第一名」
   - **類別**：例如「競賽獎勵」
   - **金額**：例如「100」
5. 點擊「✅ 記錄收入」

### 新增支出記錄

#### 預設支出類型：
- 🎁 兌換禮物
- ✏️ 購買文具
- 📚 購買書籍
- 🎮 遊戲時間
- 🍭 零食飲料

#### 操作步驟：
1. 點擊「➕ 新增記錄」
2. 選擇「🛍️ 支出」類型
3. 可以點擊預設事件快速填寫
4. 填寫金額（會自動扣除）
5. 點擊「✅ 記錄支出」

## 💡 使用場景

### 場景 1：學生參加比賽得獎 🏆

```
步驟：
1. 進入學生頁面 → 💰 獎金存摺
2. 點擊「➕ 新增記錄」

5. 修改描述：「全國數學競賽第一名」
6. 輸入金額：100
7. 點擊「✅ 記錄收入」
```

**存摺顯示**：
```
🏆 全國數學競賽第一名
競賽獎勵 | 2024/12/16          +$100
```

```

3. 選擇「🛍️ 支出」
4. 點擊「🎁 兌換禮物」
5. 修改描述：「樂高積木組」
6. 輸入金額：50
7. 點擊「✅ 記錄支出」
```

**存摺顯示**：

### 場景 3：優良表現獎勵 ⭐

```
步驟：

2. 點擊「➕ 新增記錄」

```

## 📊 與評量記錄的區別

| | 評量記錄 | 獎金存摺 |
|---|---------|---------|
| **用途** | 記錄考試成績 | 記錄特殊事件收支 |
| **獎金來源** | 🤖 自動計算（根據分數） | ✋ 手動輸入 |
| **典型場景** | 月考、小考、作業 | 得獎、購物、獎懲 |
| **關聯科目** | ✅ 必須選擇科目 | ❌ 不需要科目 |
| **記錄方式** | 輸入分數，系統計算獎金 | 直接輸入金額 |

## 🎯 最佳實踐

### ✅ 建議做法

- 及時記錄每次收支
- 使用清楚的描述（具體說明原因）
- 正確選擇類別便於統計
- 定期查看餘額

### ❌ 避免做法

- 描述過於簡略（只寫"獎金"、"花費"）
- 金額輸入錯誤
- 忘記記錄重要的收支事件

## 💡 小技巧

1. **快速選擇**：點擊預設事件可以自動填入描述和類別
2. **餘額監控**：隨時查看當前可用獎金
3. **分類統計**：使用統一的類別名稱便於未來統計
4. **即時更新**：新增記錄後餘額立即更新

## ❓ 常見問題

### Q1: 獎金存摺和評量記錄的獎金會重複計算嗎？
**答**：不會。兩者都會計入總餘額，但來源不同：
- 評量記錄：考試成績自動計算
- 獎金存摺：特殊事件手動記錄

### Q2: 可以修改或刪除存摺記錄嗎？
**答**：目前暫不支持在網頁上直接修改。如需調整，請前往 Supabase Dashboard 的 transactions 表中編輯。

### Q3: 存摺記錄的順序如何排列？
**答**：按照創建時間倒序排列，最新的記錄顯示在最上方。

### Q4: 可以自定義收支類型嗎？
**答**：可以！在「類別」欄位輸入任何自定義名稱即可。

### Q5: 負餘額怎麼辦？
**答**：系統允許負餘額，但建議您設定規則避免過度支出。
