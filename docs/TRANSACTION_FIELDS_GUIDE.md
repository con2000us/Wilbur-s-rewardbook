# 交易記錄欄位說明

## 📝 欄位差異說明

新增存摺記錄時，有兩個看似相似的欄位，但實際用途不同：

---

## 1️⃣ 事件名稱（description）

### **用途**
記錄**具體發生的事情**，這是唯一的、詳細的描述。

### **範例**

| 類型 | 事件名稱 | 說明 |
|------|----------|------|
| 收入 | `數學競賽第一名` | 具體說明是什麼競賽 |
| 收入 | `期中考進步20分` | 具體說明進步幅度 |
| 支出 | `購買鋼筆禮物` | 具體說明購買什麼 |
| 支出 | `換取遊戲時間30分鐘` | 具體說明兌換內容 |
| 歸零 | `新學期開始` | 具體說明歸零原因 |

---

## 2️⃣ 分類標籤（category）

### **用途**
用於**歸類和統計**，將相似的事件分組，方便日後查詢和分析。

### **範例**

| 類型 | 分類標籤 | 說明 |
|------|----------|------|
| 收入 | `競賽獎勵` | 所有競賽得獎都歸這類 |
| 收入 | `進步獎勵` | 所有進步獎金都歸這類 |
| 支出 | `文具購買` | 所有文具消費都歸這類 |
| 支出 | `休閒活動` | 所有遊戲時間都歸這類 |
| 歸零 | `學期更新` | 所有學期歸零都歸這類 |

---

## 🎯 對比理解

### **情境一：競賽得獎**

```
事件名稱（詳細）：數學競賽第一名
分類標籤（分組）：競賽獎勵
```

### **情境二：另一場競賽**

```
事件名稱（詳細）：英文演講比賽優勝
分類標籤（分組）：競賽獎勵  <-- 相同分類
```

**效果**：日後可以篩選「競賽獎勵」，看到所有競賽相關的記錄。

---

## 📊 實際應用

### **快速選擇的邏輯**

點擊快速選擇時，系統會自動填入：

| 快速選擇 | 事件名稱 | 分類標籤 |
|----------|----------|----------|
| 🏆 比賽得獎 | `比賽得獎` | `競賽獎勵` |
| ⭐ 優良表現 | `優良表現` | `行為獎勵` |
| 🎁 兌換禮物 | `兌換禮物` | `禮物兌換` |

然後你可以修改「事件名稱」為更具體的描述，例如：
- `比賽得獎` → `數學競賽第一名`

---

## 💡 填寫建議

### **事件名稱**
- ✅ **詳細、具體**：`數學競賽第一名`
- ✅ **包含數字**：`期中考進步20分`
- ✅ **描述內容**：`購買自動鉛筆一支`
- ❌ **太籠統**：`得獎`、`購買`

### **分類標籤**
- ✅ **簡短、統一**：`競賽獎勵`、`文具購買`
- ✅ **可重複使用**：同類事件用相同標籤
- ✅ **方便統計**：想分析時可以快速篩選
- ❌ **太細碎**：`數學競賽`、`英文競賽`（應統一為`競賽獎勵`）

---

## 🔄 歸零記錄的特殊處理

歸零記錄的「分類標籤」改為**下拉選單**，提供常用選項：

```
┌─────────────────┐
│ -- 請選擇 --    │
├─────────────────┤
│ 學期更新        │
│ 年度結算        │
│ 系統校準        │
│ 金額調整        │
└─────────────────┘
```

**原因**：歸零記錄較少，分類相對固定，下拉選單更方便。

---

## 📋 完整範例

### **收入記錄**
```
類型：💰 收入
事件名稱：數學競賽第一名
分類標籤：競賽獎勵
金額：+$100
日期：2025/01/15
```

### **支出記錄**
```
類型：🛍️ 支出
事件名稱：購買自動鉛筆一支
分類標籤：文具購買
金額：-$20
日期：2025/01/16
```

### **歸零記錄**
```
類型：🔄 歸零
事件名稱：新學期開始
分類標籤：學期更新  <-- 下拉選單選擇
日期：2025/02/01
```

---

## ✅ 總結

| 欄位 | 中文名稱 | 用途 | 特點 |
|------|----------|------|------|
| `description` | **事件名稱** | 詳細描述具體事件 | 唯一、具體、詳細 |
| `category` | **分類標籤** | 歸類和統計 | 簡短、重複、方便篩選 |

**簡單記法**：
- **事件名稱** = 這次發生了什麼（詳細）
- **分類標籤** = 這屬於哪一類（分組）
