## 🐛 Troubleshooting / 問題排除

### Q19: Deployment failed, what to do? / Q19: 部署失敗怎麼辦？

**A:** Check the following:  
**A:** 檢查以下幾點：

1. **Are environment variables correct?** / **環境變數是否正確**：

   - Confirm there are no extra spaces / 確認沒有多餘的空格

2. **Is database setup complete?** / **資料庫是否設置完成**：
   - Confirm you've run `database/bootstrap/01_schema.sql` and `database/bootstrap/02_seed_defaults.sql` in order / 確認已依序執行 `database/bootstrap/01_schema.sql` 與 `database/bootstrap/02_seed_defaults.sql`

3. **Check error messages** / **查看錯誤訊息**：
   - Check deployment logs in Vercel Dashboard / 在 Vercel Dashboard 查看部署日誌
   - Adjust according to error messages / 根據錯誤訊息調整

### Q20: Can't log into the website? / Q20: 無法登入網站？

**A:** 檢查以下幾點：
